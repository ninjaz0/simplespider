# 办公室复印纸采购优化大作业 - “谁是金牌采购官”

本项目是《Python程序设计》课程的综合性期末大作业。项目模拟了真实的商业场景：作为学校后勤办公室的首席数据采购官，在 20,000 元预算和 150 箱存储上限的限制下，利用 Python 实现数据驱动决策，以保证全年的A4复印纸供应并最大化总购买量。

项目按要求分为三大阶段：**爬虫与基础分析**、**可视化与科学建模**、**模拟实战PK**。

---

## 📂 目录结构与文件说明

| 文件名 | 阶段 | 功能说明 |
| --- | --- | --- |
| `20250001_spider.py` | 阶段一 | **真实淘宝爬虫**：基于 Selenium 自动化控制，支持手动扫码登录后抓取真实商品价格。 |
| `generate_mock_data.py`| 阶段一 | **辅助工具**：可生成高质量模拟价格数据，避免因淘宝反爬导致流程受阻。 |

| `20250001_model.py` | 阶段二 | **科学建模**：使用 Scikit-Learn (线性回归) 对清洗后的数据进行未来价格预测。 |
| `20250001_decision.py` | 阶段三 | **采购决策系统**：严格遵守题目各项约束条件，根据输入价格动态计算最优采购量的交互程序。 |

## 🛠️ 环境依赖与安装

本项目基于 **Python 3.8+** 开发，运行前请确保安装以下核心依赖库：

```bash
# 安装基础数据分析与机器学习库
pip install pandas numpy matplotlib seaborn scikit-learn

# 安装真实爬虫所依赖的自动化控制库
pip install selenium webdriver-manager
```

*注意：第一阶段的淘宝爬虫程序 `20250001_spider.py` 需要您的电脑上安装有 **Google Chrome** 浏览器。*

---

## 🚀 快速开始与运行指南

请按照以下顺序执行脚本，以体验完整的数据流闭环：

### 1. 数据采集 (阶段一)
如果您希望体验**真实淘宝爬取**：
```bash
python 20250001_spider.py
```
> **提示**：运行后会自动打开 Chrome 浏览器，请在 60 秒内使用手机淘宝扫描二维码登录。登录后程序将自动接管页面进行搜索和抓取，并保存为 `raw_scraped_data.csv`。

如果您希望使用**安全稳定的大批量模拟数据**：
```bash
python generate_mock_data.py
# 将自动生成包含一年价格波动的 raw_paper_data.csv
```

### 2. 数据预处理 (阶段一)
清洗原始数据并输出满足 PK 赛格式要求的最终文件：
```bash
python 20250001_preprocess.py
# 产出文件：20250001_result.csv
```

### 3. 数据可视化 (阶段二)
生成价格走势图、箱线图、热力图等分析图表：
```bash
python 20250001_visual.py
# 产出文件：4张高清 png 图片
```

### 4. 价格预测建模 (阶段二)
训练线性回归模型并预测未来商品价格：
```bash
python 20250001_model.py
# 控制台将输出 RMSE 与 MAE 评估指标
# 产出文件：拟合效果图 及 预测结果表 20250001_predict.csv
```

### 5. 实战采购决策PK (阶段三)
启动核心打分环节的交互式购买决策系统：
```bash
python 20250001_decision.py
```
程序启动后，会首先输出 `5.1,20` (首月扣减情况)。
随后请根据提示输入测试用例（如：`6.20,82,齐心白云海`），程序将瞬间输出最优的购买与库存核减方案。

## 📝 核心策略简述
第三阶段的购买策略采用了**带安全库存检查的贪心囤货策略**：
1. **强制合规**：彻底贯彻预算不超 20000、库存不超 150、绝不断货、次年5月保底 20 箱的红线。
2. **极佳价格（≤80元）**：触发满仓买入逻辑，在预算允许下打满 150 箱库存。
3. **良性价格（81-85元）**：触发半仓买入逻辑，囤货的同时保留预算防御未来的未知高价。
4. **紧急避险（>85元）**：除非系统预测未来两个月即将断货而被迫进行紧急小额采购，否则坚决不买。
